Welcome to CM Short URL's documentation!
====================================================

CM Short URL is a Joomla! component for shortening URL, it also provides option to add Google Analytics tracking parameters to URL before shortening and option to log visits from shortened URLs.

This documentation gives you instruction how to install and setup CM Short URL on your Joomla! website.
